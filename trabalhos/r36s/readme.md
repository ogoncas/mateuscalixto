# R36S Hub

R36S Hub, uma plataforma comunitária dedicada a temas, tutoriais e firmwares para o dispositivo R36S. Este projeto é uma biblioteca colaborativa para entusiastas de emulação retro.

## Sobre

O R36S Hub é uma plataforma web desenvolvida para a comunidade de usuários do R36S, um handheld de emulação retro. Aqui, você pode explorar uma vasta coleção de temas personalizados, tutoriais detalhados e firmwares atualizados. A comunidade pode contribuir com seus próprios projetos, favoritar conteúdos e gerenciar publicações através de uma interface amigável.

- **Tecnologias utilizadas**: HTML5, CSS3, JavaScript, Firebase (para autenticação e armazenamento de dados), Marked.js (para renderização de Markdown).
- **Autor**: Mateus Calixto.
- **Versão**: 0.1 (lançada em 2026).
- **Objetivo**: Promover o compartilhamento de recursos e fomentar a colaboração na comunidade retro gaming.

Para mais detalhes, visite o site: [R36S Hub](https://mateuscalixto.com.br/trabalhos/r36s).

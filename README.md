# 🚀 Mateus Calixto | Software Developer Portfolio

Este é o repositório do meu portfólio pessoal, projetado para apresentar meus projetos e habilidades técnicas como desenvolvedor Front-end e especialista em Firebase.

![Preview do Projeto](https://mateuscalixto.com.br/favicon.png) 

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3 Moderno:** Estruturação semântica e layouts baseados em Flexbox e Grid.
* **JavaScript (ES6+):** Lógica para animações de scroll, controle de menu mobile e efeitos de parallax 3D.
* **Firebase:** Utilizado para hospedagem (Hosting) e arquitetura serverless.
* **Design:** Tipografia com fontes 'Outfit' e 'Inter', e paleta de cores 'Midnight'.

## ✨ Funcionalidades

* **📱 Responsividade Total:** Adaptado para dispositivos mobile, tablets e desktops.
* **🖱️ Hero Parallax:** Efeito visual interativo no card de código que segue o movimento do mouse.
* **📜 Scroll Reveal:** Animações de entrada suaves conforme o usuário navega pela página.
* **🧭 Navegação Inteligente:** Header dinâmico com efeito blur e links ativos durante o scroll.

## 📁 Estrutura do Projeto

* `index.html`: Estrutura principal e metadados de SEO/Open Graph.
* `style.css`: Design system, variáveis, animações e media queries.
* `script.js`: Toda a interatividade e otimizações de performance (debounce/throttle).
